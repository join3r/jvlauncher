## [Unreleased]

### Added
- Initial release of App Launcher
- Global shortcut to show/hide launcher
- Support for three app types:
  - Native applications with command-line parameters
  - Web applications with persistent sessions
  - Terminal applications with embedded terminal
- Grid layout with customizable size
- Keyboard navigation (arrow keys, enter, escape)
- Drag and drop reordering
- Icon extraction from applications
- Custom icon support
- Per-app keyboard shortcuts
- Settings panel:
  - Theme selection (system/light/dark)
  - Grid size configuration
  - Start at login option
  - Global shortcut customization
- Right-click context menu for edit/delete
- SQLite database for data persistence
- Cross-platform support (macOS, Linux, Windows)
