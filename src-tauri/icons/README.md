# Icon Assets

Place your application icons here:

- `32x32.png` - 32x32 pixel icon
- `128x128.png` - 128x128 pixel icon
- `128x128@2x.png` - 256x256 pixel icon (for Retina displays)
- `icon.icns` - macOS icon file
- `icon.ico` - Windows icon file

You can generate these from a single source image using online tools or icon generation software.

For development, placeholder icons will be used if these files are not present.
